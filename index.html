Saya, Muhammad Fawwaz, adalah seorang Software Engineer dengan 3 tahun pengalaman dalam membangun aplikasi web yang berorientasi pada kinerja dan pengalaman pengguna. Perjalanan teknis saya dimulai dari HTML/CSS/JS dasar, kemudian fokus pada ekosistem React.js dan pengembangan API dengan Node.js/Express di sisi server.

Selama karir, saya telah berhasil memimpin inisiatif migrasi dari arsitektur monolitik ke Microservices menggunakan Docker dan Kubernetes, yang menghasilkan peningkatan deployment speed* sebesar 40%. Filosofi saya adalah "Code as Clean as Possible." Saya berpegang teguh pada prinsip-prinsip SOLID dan selalu mengimplementasikan TDD (Test-Driven Development) untuk menjamin kualitas dan daya tahan kode. Saat ini saya bekerja sebagai Senior Full-Stack Developer di Tech Innovate Solutions.
